## np.insert()用法
numpy.insert可以有三个参数（arr，obj，values），也可以有4个参数（arr，obj，values，axis）  
第一个参数arr是一个数组，可以是一维的也可以是多维的，在arr的基础上插入元素  
第二个参数obj是元素插入的位置  
第三个参数values是需要插入的数值  
第四个参数axis是指示在哪一个轴上对应的插入位置进行插入
